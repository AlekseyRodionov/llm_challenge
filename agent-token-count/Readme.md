# agent-token-count

CLI-агент для взаимодействия с LLM с детальным подсчетом токенов.

---

## Описание

`agent-token-count` — это CLI-инструмент для интерактивного общения с LLM.  
Показывает детальную статистику токенов: запрос, история, ответ, лимиты.

## Возможности

- Интерактивный чат с LLM
- Детальный подсчет токенов (запрос, история, ответ)
- Отслеживание лимита модели с поправочным коэффициентом
- Сохранение контекста между сессиями
- Демонстрация переполнения токенов

## Установка

```bash
cd agent-token-count
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка

```bash
cp .env.example .env
```

```env
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=openai/gpt-4o-mini
```

## Использование

```bash
python -m app.main
```

### Команды

| Команда | Описание |
|---------|----------|
| `help` | Показать справку |
| `stats` | Показать статистику сессии |
| `reset` | Очистить историю |
| `history` | Показать историю |
| `tokens` | Показать текущие токены |
| `fill` | Добавить ~80K мусорных токенов |
| `exit` | Выйти |

### Пример вывода

```
Вы: привет

Агент: Привет! Как я могу помочь?

Токены:
  Запрос: 6
  История: 11
  Ответ:   10
  ─────────────
  Всего:   27

Лимит:
  Использовано: 17 / 128000 (0.0%)
  Осталось: 127983
  Стоимость: 0.000828 ₽
```

### Демонстрация переполнения

```bash
# Очищаем историю
reset

# Начинаем диалог
привет

# Добавляем мусорные токены
fill
fill

# Отправляем запрос - получаем ошибку
продолжим
```

Вывод:
```
⚠ ПРЕВЫШЕН ЛИМИТ!
Ошибка от LLM: This endpoint's maximum context length is 128000 tokens...
```

## Метрики токенов

- **Запрос** — токены текущего сообщения пользователя
- **История** — токены всех предыдущих сообщений
- **Ответ** — токены от модели
- **Лимит** — gpt-4o-mini имеет 128K токенов

### Поправочный коэффициент

Для более точного соответствия с LLM API используется коэффициент **1.55**,
учитывающий разницу между tiktoken и внутренним подсчетом OpenAI.
